1. 웹 브라우저 : request >> "index.html을 주세요"

- 웹 브라우저 >> 요청(request) : 클라이언트(client)
- 웹 서버 >> 응답(response) : 서버(server)

- 웹 호스팅 vs 웹 서버
    - 업체의 서버를 사용 > 쉽지만, 블랙박스
    - 웹 서버 설치 > 어렵지만, 지식&정보
